Tomcat 7 Servlet + JSP Maven template
=====================================

A template for a standard Servlet 3.0 + JSP Maven project

[Maven home page](https://maven.apache.org)

Maven Commands
______________

    mvn clean

Cleans application

    mvn tomcat7:run

Runs application

    mvn package

Builds WAR

    mvn clean package

Goals can be combined
